# LaTeX_pattern
Шаблон для создания LaTeX документов
- --

В папке *doc* - файлы с документацией к моей преамбуле.

В папке *preamble* - собственно сама преамбула.

### Для создания нового документа необходимо:

1) Создать папку, в которой будет создан документ;
2) Переместить в неё папку *preamble*;
3) Начать создавать документ согласно документации из *doc*.
