Абсолютные единицы

    pt (point) — пункт (1 pt = 1/72.27 дюйма).
        Пример: 1pt
        Часто используется для шрифтов и точных отступов.

    in (inch) — дюйм (1 in = 2.54 см).
        Пример: 1in
        Удобен для задания размеров, соотносящихся с реальными физическими величинами.

    cm (centimeter) — сантиметр.
        Пример: 1cm
        Используется в метрической системе, широко применяемой в Европе.

    mm (millimeter) — миллиметр.
        Пример: 1mm
        Подходит для очень мелких отступов.

    bp (big point) — большой пункт (1 bp = 1/72 дюйма).
        Пример: 1bp
        Используется для графики и шрифтов.

    pc (pica) — пика (1 pc = 12 pt = 1/6 дюйма).
        Пример: 1pc
        Традиционная единица для типографов.

    dd (didot point) — пункт Дидо (1 dd = 1.07 pt).
        Пример: 1dd
        Используется в старой европейской типографике.

    cc (cicero) — цицеро (1 cc = 12 dd).
        Пример: 1cc
        Европейский аналог пики.

Относительные единицы

    em — ширина буквы "M" в текущем шрифте.
        Пример: 1em
        Используется для пропорциональных отступов и размеров шрифтов.

    ex — высота строчной буквы "x" в текущем шрифте.
        Пример: 1ex
        Часто применяется для вертикальных отступов.

    mu — математическая единица (1mu = 1/18 em в текущем математическом шрифте).
        Пример: 1mu
        Применяется в математических формулах для точного позиционирования.

Другие полезные единицы

    \textwidth — ширина текста.
        Пример: 0.5\textwidth
        Используется для задания размеров элементов относительно ширины текстового блока.

    \linewidth — ширина текущего блока (например, в среде minipage).
        Пример: \linewidth
        Полезно при работе с блоками текста, отличающимися от основного текста по ширине.

    \paperwidth и \paperheight — размеры страницы.
        Пример: \paperwidth
        Применяется для позиционирования элементов на всей странице.

    \baselineskip — межстрочный интервал.
        Пример: 0.5\baselineskip
        Используется для вертикальных отступов между строками.

Смешанные единицы

    \hspace{<длина>} — горизонтальный пробел.
        Пример: \hspace{1cm}
        Позволяет создать отступ определенной длины.

    \vspace{<длина>} — вертикальный пробел.
        Пример: \vspace{1em}
        Используется для создания вертикального пространства.
